# bmmc


bmmc is a Python wrapper for the Minio _mc_ command line interface.

## Installation

```bash
pip install bmmc
```

## Development

```bash
git clone https://gitlab.com/big-mama-tech/bmmc
cd bmmc
pip install -e .

# Generate documentation
sphinx-build -b html docs docs/html
```
